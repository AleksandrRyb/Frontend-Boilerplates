## 🍩 Starting the project
- Get the boilerplate
> [Download ZIP](https://github.com/CKGrafico/Frontend-Boilerplates/archive/nodets.zip)

- Install dependencies:
> npm install

- Running tasks on watch mode and serve the app:
> npm start
